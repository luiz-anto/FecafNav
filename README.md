FecaNav
==============
Este projeto foi desenvolvido para apresentação no evento Expotech 2025, da UniFecaf.

Trata-se explicitamente de uma modificação do projeto original disponível em:
https://github.com/cykod/AlienInvasion

Todo o desenvolvimento deste repositório foi realizado tomando o projeto original como base, adaptando-o e expandindo-o conforme as necessidades da exposição.

Este projeto não tem qualquer intenção de atacar, prejudicar ou ofender qualquer pessoa, entidade ou instituição.

Tecnologias utilizadas:

HTML5

CSS3

JavaScript

O FecaNav foi otimizado principalmente para uso em navegadores mobile.
